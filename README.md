# Golfive Website

A professional, modern golf blog website built with HTML5, CSS3, and JavaScript. Designed with a mobile-first responsive approach and future-ready for e-commerce integration.

## Features

- **Modern Design**: Clean, professional golf-themed design with gradient color scheme
- **Mobile-First Responsive**: Fully responsive across all device sizes
- **Performance Optimized**: Lazy loading, efficient CSS, and modern JavaScript
- **SEO Friendly**: Semantic HTML5 structure and proper meta tags
- **Accessible**: WCAG compliant with proper ARIA attributes
- **Modular Architecture**: Component-based CSS structure ready for scaling
- **Future E-commerce Ready**: Built with extensibility in mind

## Tech Stack

- **HTML5**: Semantic markup structure
- **CSS3**: Custom properties, flexbox, grid, and modern layout techniques
- **JavaScript (ES6+)**: Modern features with progressive enhancement
- **Responsive Design**: Mobile-first approach with comprehensive breakpoints
