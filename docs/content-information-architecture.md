# Golfive 内容信息架构（V1）

## 品牌定位
- 实战导向、可执行、少术语堆砌
- 目标：帮助普通球友稳定降杆，而不是追求花哨动作

## 三条内容主线
- A 技巧（50%）
  - 热身、短杆、推杆、开球稳定、心态
- B 装备（30%）
  - 新手预算、挥速分组、选杆避坑、评测逻辑
- C 球场与策略（20%）
  - 球场攻略、赛前准备、路线选择、风险管理

## 用户分层
1. 新手（0-12个月）
2. 进阶（90-100杆区间）

## 内容漏斗
- 顶部（流量）：技巧类问题解答
- 中部（信任）：装备选择框架 + 球场策略
- 底部（转化）：Newsletter / Contact / 深度专题

## 页面内链路径（示例）
- 热身文章 -> 短杆文章 -> 推杆文章 -> Newsletter
- 装备文章 -> 驱动杆评测 -> 联系咨询
- 球场攻略 -> 赛前清单 -> Newsletter

## 首批重点落地
1) 下场前10分钟热身流程
2) 短杆100码稳定打法
